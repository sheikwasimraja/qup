Quran Progress Tracker Guide

This guide explains how to use the tracker well, keep progress organised, and make the most of each companion page.

Why Keep Going

Learning the Quran is a journey of patience, consistency, and barakah. Small daily steps matter. A tracker can help you return to the Quran with clarity, measure progress honestly, and stay encouraged even when growth feels slow.

1. What This Version Includes

  • quran-progress-tracker.html is the main tracker.
  • quran-progress-summary.html shows progress overview charts.
  • quran-progress-trends.html shows daily and monthly trends.
  • quran-progress-motivation.html gives space for your purpose, reminders, and encouragement.
  • quran-reference-settings.html controls reference links and visible columns.
  • Desktop shows the full table, while smaller screens switch to a mobile cards view for easier reading and tapping.
  • Your verse progress is saved on your own device using browser storage, with a fallback when some mobile browsers are more limited.

2. Daily Use

  1. On desktop, search, filter, or use the header filters under each column. On mobile, use the card filters at the top of the cards view.
  2. Use the LEARNT and MEMORISE toggle buttons row by row or card by card. One tap marks Yes; tapping again clears it.
  3. When you mark LEARNT as Yes, the tracker fills DATE LEARNT automatically.
  4. When you mark MEMORISE as Yes, the tracker fills DATE MEMORISED automatically.
  5. Click Save Progress to save your current work on the device you are using.
  6. Click Download Backup from time to time to keep a JSON copy, and use Restore Backup if you need to bring that saved copy back.
  7. If you want to discard unsaved edits from your current session, use Reset Unsaved Changes.

3. Reviewing And Staying Motivated

  1. Use Random Review Verse when you want a quick revision prompt.
  2. Use Show Context after a random review verse when you want to open nearby verses for more context.
  3. Use Today's Progress to see what you completed on the current day.
  4. Open the motivation page whenever you want to revisit your intention and reminders.
  5. Check the summary and trends pages to see how small efforts are adding up over time.

4. Settings And References

  1. Open Reference Settings from the tracker.
  2. Choose which reference sources should open.
  3. Choose which columns should be visible.
  4. Save the settings and refresh the tracker if it is already open.

Features Available

  • Desktop table with search, paging, and column-wise filters.
  • Mobile cards view with compact filters and card-by-card progress controls.
  • One-tap LEARNT and MEMORISE toggle buttons instead of dropdowns.
  • Random Review Verse to pick a random verse not yet marked as learnt.
  • Show Context to open the surrounding verses for a review selection.
  • Today's Progress to show rows learnt today.
  • Updated verses to show only the rows changed in your current session.
  • A motivation page for intention-setting, reminders, and encouragement.
  • Open all references to open the selected reference websites for a verse.
  • Restore Backup to load a previously downloaded JSON backup.
  • Reset Unsaved Changes to clear edits that have not been saved yet.
  • Close Reference Tabs to close reference tabs opened by the tracker when the browser allows it.
  • Summary and trend pages that reflect your current saved progress on that device.

How Saving Works

Your progress is saved on your own device in browser storage. If another person opens the same site in their own browser, they will keep their own separate progress. Some mobile browsers may use a browser storage fallback behind the scenes, so it is still a good idea to download a backup file from time to time in case browser storage is cleared or limited.

Sharing With Someone Else

  1. Share the website or repository link.
  2. Each person can use the same site independently.
  3. Each browser keeps its own saved progress.
  4. No one overwrites another person's learning progress unless they share the same browser storage.

Troubleshooting

  • If the tracker looks old, refresh the page once.
  • If charts do not match the tracker, save your progress and refresh the chart page.
  • If settings do not appear, save them and then refresh the tracker.
  • If the tracker looks different on a phone, that is expected because smaller screens switch to the mobile cards view.
  • If a reference tab does not close, that website may block script-based closing after it opens.
  • If saving fails on a device, use Download Backup right away so you still have a copy of your progress.
  • If browser data is cleared, the tracker returns to its original seeded data unless you restore from a backup.
